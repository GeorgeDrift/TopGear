
# Top Gear Motors Malawi - Project Documentation

## Project Overview
Top Gear Motors is a premier vehicle marketplace designed for the Malawi automotive market. It features a multi-role architecture supporting Administrators, Dealership Managers, Sales Agents, and Buyers.

## Technical Stack
- **Frontend**: React (ES6 Modules)
- **Styling**: Tailwind CSS (Mobile-First Responsive Design)
- **Icons**: Lucide React
- **Intelligence**: Gemini 3 Flash Preview (Dynamic Inventory Generation & Stock Management)
- **Backend Architecture**: Modular View Pattern with Express/Node integration points
